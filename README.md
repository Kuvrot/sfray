# sfray
Old-school FPS raycast engine written in C++ and using SFML.

# Raycaster

## Rendering Options
There are different rendering "options" for walls, floor, ceiling and entities.

### Walls

- None
- Colored
- Textured

### Floor

- None
- Colored
- Textured

### Ceiling

- None
- Colored
- Textured

### Entities

- None
- Textured
